# Session 1 - ETL Extraction Layer

## Objective

Implement only the ETL extraction layer for Instagram and WhatsApp JSON exports.

## Completed

- Created `etl/extract.py`.
- Implemented safe JSON file loading.
- Implemented dynamic JSON schema inspection.
- Implemented Instagram JSON structure detection.
- Implemented WhatsApp JSON structure detection.
- Returned raw pandas DataFrames without transformation.
- Logged file, JSON parsing, unsupported schema, empty record, and runtime extraction errors through `logger/logger.py`.
- Kept database access outside ETL extraction.
- Created `tests/test_extract.py`.
- Created `scripts/inspect_json_schema.py` for manual schema debugging.
- Updated `docs/context.md` and `docs/tasks.md`.

## Verification

Run:

```bash
pytest tests/
```

Manual schema inspection:

```bash
python scripts/inspect_json_schema.py data/raw/example.json

python scripts/inspect_json_schema.py data/raw/ig_message_2.json


```
