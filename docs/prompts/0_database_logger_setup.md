# Session 0 Prompt: Database Logger Setup

Implement the initial infrastructure layer only:

- PostgreSQL schema in `db/schema.sql`.
- Database connection helpers in `db/connection.py`.
- Parameterized SQL queries in `db/queries.py`.
- Centralized file and database logger in `logger/`.
- Manual scripts in `scripts/`.
- Unit tests in `tests/`.
- Environment example and documentation updates.

Do not implement ETL logic, analysis logic, NLP, metrics, sentiment, or Streamlit dashboard UI.
